# **เว็บแสดงการ์ดที่สแกนเวลาเข้าห้อง**
1.แสดงห้อง และการ์ดของห้องนั้น
2.แสดงเวลาที่สแกนของการ์ดในนั้น ว่าสแกนเวลาไหน แล้วเหลือการ์ดใบไหนบ้างที่ยังไม่ได้สแกน
